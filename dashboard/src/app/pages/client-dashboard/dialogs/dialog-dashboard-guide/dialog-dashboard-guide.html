Dashboard Guide

To get the most out of your dashboard, follow these steps:

1. Select a Date Range

Choose a specific time period or pick a single day to view trends within that range.

2. Use Filters for Precision

Filters allow you to drill down into the details. If you multi-select sites, you won’t be able to use filters for Floor, Area, Location, or Outlet Type—these require a single site selection.

3. Filters and Parameters

  • Outlet Type: This filter will only become active once you select at least one site.
    • Single Parameter Graphs: These will only appear when you select a single outlet.
    • Multi-site Selection: If you choose multiple outlet sites, the single parameter graphs will be hidden, but you’ll be able to compare passed and failed results across sites.
