# aqua-protrack

Required applications

=====================================

1. Docker desktop
2. Visual Studio Code





Starting Supabase

=====================================

Ensure Docker is running

Navigate to the supabase location

npx supabase start -> Starts supabase

npx supabase stop -> Stops supabase

Different links

http://127.0.0.1:54323 -> Supabase UI

http://127.0.0.1:54324 -> Inbucket UI, for email viewing





Registering edge function

=====================================

npx supabase functions serve register-admin-user





Starting Application

=====================================

Navigate to the dashboard location

npm i -> Only need to do this the first time

npm start -> Start the angular application.
